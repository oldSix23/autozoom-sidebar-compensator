# 商店发布信息

## Edge Add-ons

### 发布地址
https://partner.microsoft.com/dashboard/microsoftedge/

### 基本信息
- **名称**: AutoZoom - Sidebar Width Compensator
- **版本**: 1.1.0
- **类别**: Productivity
- **语言**: English, 中文

### 英文描述 (English)
Tired of websites being cut off by your browser's vertical tabs or sidebar?

AutoZoom automatically detects the sidebar width and scales the page to fit your actual viewport — no manual adjustment needed.

**Features:**
- Auto-detect sidebar/vertical tabs width and compensate with smart zoom
- Real-time adjustment when resizing sidebar or window
- Per-site blacklist for sites you don't want to zoom
- Manual zoom detection — pauses auto-zoom when you Ctrl+/- zoom
- Quick toggle with Alt+Z keyboard shortcut
- Badge indicator showing current zoom percentage
- Smooth zoom transition animation
- Bilingual UI (English/Chinese, auto-detected)
- Zero configuration, works out of the box

**Works with:**
Edge vertical tabs, Edge sidebar, Chrome side panel, Brave sidebar, Opera sidebar, and any browser feature that reduces your viewport width.

### 中文描述
使用垂直标签页或侧边栏时，网页显示不全？AutoZoom 自动检测侧边栏宽度，智能缩放页面，让内容完整显示，无需手动调整。

**功能特点：**
- 自动检测侧边栏/垂直标签页宽度，智能计算缩放比例
- 拖动侧边栏或调整窗口大小时实时响应
- 支持按站点设置黑名单，不想缩放的站点一键排除
- 智能识别手动缩放，避免与 Ctrl+/- 冲突
- Alt+Z 快捷键一键开关
- 角标显示当前缩放比例
- 平滑过渡动画
- 中英文自动切换
- 开箱即用，无需配置

**适用场景：**
Edge 垂直标签页、Edge 侧边栏、Chrome 侧面板、Brave 侧边栏、Opera 侧边栏等所有占用页面宽度的浏览器功能。
